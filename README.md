# MiniGit

A simplified Git version control system built in C++ to demonstrate the core concepts behind Git, including version tracking, branching, commits, and merging.

---

## Features

- Initialize a repository
- Add files to the staging area
- Create commits
- View commit history
- Create branches
- Switch between branches
- Merge branches
- Compare file differences

---

## Technologies Used

- C++
- C++17 Filesystem Library
- File Handling
- Object-Oriented Programming (OOP)

## How to Run

Compile:

```bash
g++ -std=c++17 main.cpp -o MiniGit
```

Run:

```bash
MiniGit.exe
```

---

## Example Commands

```
init
add test.txt
commit Initial commit
log
branch feature
checkout feature
merge feature
diff
exit
```
